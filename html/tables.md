# 2.12 表格

基本結構 1

```html
<table>
  <tr>
    <td></td>
    <td></td>
  </tr>
</table>
```

基本結構 2

```html
<table>
  <tr>
    <th></th>
    <th></th>
  </tr>
  <tr>
    <td></td>
    <td></td>
  </tr>
</table>
```

進階結構

```html
<table>
  <thead>
    <tr>
      <th></th>
      <th></th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td></td>
      <td></td>
    </tr>
  </tbody>
</table>
```
