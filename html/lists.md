# 2.8 列表

## 有序列表 - &lt;ol&gt;...&lt;/ol&gt;

`<ol>`：**o**rder **l**ist

`<ol>` 裡的第一層子元素，一定是 `<li>...</li>`

`<li>`：**l**ist **i**tem

語意：有序性的列表項目。

範例：

```markup
<ol>
  <li>第一項</li>
  <li>第二項</li>
  <li>第三項</li>
</ol>
```

## 無序列表 - &lt;ul&gt;...&lt;/ul&gt;

`<ul>`：**u**norder **l**ist

`<ul>` 裡的第一層子元素，一定是 `<li>...</li>`

`<li>`：**l**ist **i**tem

語意：**無**有序性的列表項目。

範例：

```markup
<ul>
  <li>第一項</li>
  <li>第二項</li>
  <li>第三項</li>
</ul>
```

結果呈現：
