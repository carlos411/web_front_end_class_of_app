# 2.17 HTML Entity\(實體\)

如遇特殊符號，可嚐試使用 HTML Entity 來顯示，例：

```html
<body>
  &lt;
  <br>
  &gt;
  <br>
  &copy;
  <br>
  &quot;
  <br>
  &apos;
  <br>
  &amp;
</body>
```
