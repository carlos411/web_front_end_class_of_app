# 2.11 圖片 - &lt;img&gt;

`<img src="#">`

`<img>` 為 **im**a**g**e 的縮寫。

## 語意

圖片的意思

## 常用屬性

```html
<img src="#" alt="圖片未正常出現會出現的文字" title="滑鼠移過，稍微停留會出現的文字">
```

## 範例 1

正常圖片顯示，以及滑鼠移過後，出現 **title** 的文字。

```html
<img src="http://notes.carlos-studio.com/images/butterfly.png" alt="圖片未正常出現會出現的文字" title="滑鼠移過，稍微停留會出現的文字">
```
