# 2.4.1 中介資料\(Meta Data\) - &lt;meta&gt;

## 語意

該網頁的額外資料，會被瀏覽器解讀。

## 範例

```html
<head>
  <meta charset="utf-8">
</head>
```

## 其他範例

```html
<head>
  <meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">
  <meta name="description" content="網站前端教學">
  <meta http-equiv="refresh" content="30">
</head>
```
