# 第一個網頁

建立一個專案資料夾，名稱自取，裡面要有以下幾個項目：

* css/
* fonts/
* images/
* js/
* index.html

編輯 index.html：

## 第一步：撰寫 html

```html
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>這是網頁標題</title>
  </head>
  <body>
    <h1>這是內文標題</h1>
  </body>
</html>
```

## 第二步：撰寫 CSS

```css
<style>
h1{
  color: red;
}
</style>
```
