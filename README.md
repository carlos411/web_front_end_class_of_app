# 1 簡介

網站前端開發，核心技術為 **HTML**、**CSS**、**JavaScript**，從這三個前端技術開始紮根，未來可應用至其他領域\(例：網頁動畫、元件、資料視覺化等\)。故 HTML、CSS、JavaScript 是網站開發中一切的基礎。

那這三者又分別代表什麼意義呢？

# 1.1 HTML = HyperText Markup Language

HTML 負責「**結構**」

# 1.2 CSS = Cascading Style Sheets

CSS 所扮演的角色為「**外觀**」，可以美化頁面，

# 1.3 JS = JavaScript

JavaScript 負責「**行為**」
