# 4.1.4 加減乘除及初探 Console

`+`：加法。

`-`：減法。

`*`：乘法。

`/`：除法。

`%`：取餘數。

## 範例

```js
var a = 5;
var b = 2;

var add = a + b;
var minus = a - b;
var multiple = a * b;
var divide = a / b;
var remainder = a % b;
```

## 練習

試試看瀏覽器中的 **Console\(控制台\)**。

以 Mac 電腦為例，使用 **Chrome 瀏覽器**，快速鍵 `cmd + option + i`，即可看到 **console** 的頁籤。
