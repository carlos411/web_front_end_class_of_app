# 4.1.5 邏輯\(logic\)

**布林值\(Boolean\)**：**true** 或 **false**

```js
var a = true;
var b = false;
```

## 比較運算子\(Operator\)

`===`：「值」、「型態」都要一樣，才會視為 **true**。例：`10 === "10"`，結果為 **false**。

!==：「值」、「型態」都要不一樣，才會視為 **true**。例：`10 !== "10"`，結果為 **true**。

`==`：「值」只要一樣，結果即會被視為 **true**。

`!=`：「值」只要不一樣，結果即會被視為 **true**。

`>`：大於。例： `5 > 3`，結果為 **true**。

`<`：小於。例：`5 < 3`，結果為 **false**。

`>=`：大於等於。例：`5 >= 3`，結果為 **true**。

`<=`：小於等於。例：`5 <= 3`，結果為 **false**。
