# 4.1.5 邏輯\(logic\)

**布林值\(Boolean\)**：**true** 或 **false**

```js
var a = true;
var b = false;
```

## 比較運算子\(Operator\)

`===`：「值」、「型態」都要一樣。才會視為 true。

!==：「值」、「型態」都要不一樣。才會視為 true。

`==`：「值」只要一樣，即會被視為 true。

`!=`：「值」只要不一樣，即會被視為 true。

`>`：大於。

`<`：小於。

`>=`：大於等於。

`<=`：小於等於。
