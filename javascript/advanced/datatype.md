# 4.3.1 資料格式：XML 與 JSON

無論是同個系統或是多個系統之間，互相傳遞資料時，最常用的兩種格式，即 **XML** 與 **JSON**。都是純文字格式。

* XML = e**X**tensible **M**arkup **L**anguage。HTML 就是 XML 的其中一種。
* JSON = **J**ava**S**cript **O**bject **N**otation。
